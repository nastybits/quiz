# WebQuiz

Web-приложение для создания и проведения игр в жанре Квиз.

## Конфигурация

Смотри [Vite Configuration Reference](https://vitejs.dev/config/).

## Развертывание

```sh
npm install
```

### Сборка Dev версии

```sh
npm run dev
```

### Сборка Prod версии

```sh
npm run build
```

### Линтер [ESLint](https://eslint.org/)
```sh
npm run lint
```
